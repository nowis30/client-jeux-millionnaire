- API (Render) : https://server-jeux-millionnaire.onrender.com

## Configuration

- NEXT_PUBLIC_API_BASE doit pointer vers l’API Render en prod.
	- Exemple: `NEXT_PUBLIC_API_BASE=https://server-jeux-millionnaire.onrender.com`
